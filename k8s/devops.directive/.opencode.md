# DevOps Directive: Kubernetes Course

## Project Overview
This project follows the [DevOps Directive Kubernetes Course](https://github.com/sidpalas/devops-directive-kubernetes-course).

## Tech Stack (from devbox.json)
- **Cluster Orchestration:** `kind`, `kubectl`
- **Package Management:** `helm`, `kustomize`
- **Development Workflow:** `tilt`, `ko`, `go-task`
- **Cloud/Infrastructure:** `civo`, `gcloud`, `oras`
- **Languages/Runtimes:** `Go`, `Node.js 20`, `Python 3.12`

## Learning Roadmap
1. [x] **Environment Setup:** Install `devbox` and initialize the shell.
2. [ ] **Container Fundamentals:** Building images with `ko` and `docker`.
3. [x] **Local Cluster Operations:** Managing clusters with `kind`.
4. [ ] **Kubernetes Objects:** Understanding Pods, Deployments, Services, and Ingress.
5. [ ] **Configuration Management:** Using `Helm` and `Kustomize`.
6. [ ] **Continuous Development:** Rapid iteration with `Tilt`.
